🍽️ Restaurant Billing System – Python

A simple command-line Python project that simulates a restaurant ordering system.

✅ Features
Menu stored using a dictionary
User can order 1 or multiple items
Validates item availability
Adds item price to total
Displays the final bill amount
🧰 Concepts Used
Python dictionaries
Conditional statements
Looping & user input
String formatting
Basic arithmetic operations
🎯 Purpose

This project was built to practice Python fundamentals and understand how real billing systems work logically.
