✨ Today I built a simple Restaurant Billing System using Python!
This mini-project helped me practice important programming concepts like:

🍕 Using dictionaries to store menu items and prices
🧠 Working with conditional statements (if-else)
⌨️ Taking user input
➕ Updating totals using arithmetic operations
🔁 Handling multiple orders using logical flow

✔ What the program does:
Shows a menu with item prices
Lets the user order one or more items
Checks if the ordered item exists in the menu
Adds the price to the total bill
Displays the final payable amount
💡 Key Concepts Practiced:
Python dictionary operations
String formatting with f-strings
Input validation
Basic decision-making logic

This was a fun way to apply Python fundamentals to a real-world scenario!
